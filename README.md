# AutomatizacionRPA_Stocker
Solución híbrida de hiperautomatización de stock (patrón Dispatcher-Performer). Integra n8n para la orquestación asíncrona distribuyendo eventos hacia Power Automate Cloud, gestión de colas de trabajo en Dataverse y ejecución en local con Power Automate Desktop (scraping web). Base de datos PostgreSQL en Supabase.
